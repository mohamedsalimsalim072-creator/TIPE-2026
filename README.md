# TIPE – Détection d’anomalies dans un réseau informatique

## Sujet

Comment détecter automatiquement un comportement anormal dans un réseau informatique ?

Ce TIPE s’inscrit dans le thème **Sobriété – Efficacité – Optimisation**.

## Contexte

Les réseaux informatiques transportent aujourd’hui une grande quantité de données. Ils peuvent être la cible d’attaques ou de comportements anormaux comme les scans de ports, les attaques par déni de service ou les communications inhabituelles entre machines.

Détecter ces anomalies rapidement est essentiel pour garantir la sécurité et le bon fonctionnement des infrastructures numériques.

## Problématique

Comment concevoir un algorithme capable de détecter automatiquement des anomalies dans le trafic réseau tout en minimisant :

* le temps de calcul
* l’utilisation des ressources
* le nombre de fausses alertes ?

## Objectifs du projet

* Modéliser le trafic d’un réseau informatique.
* Identifier des indicateurs permettant de caractériser un comportement normal.
* Concevoir un algorithme de détection d’anomalies.
* Comparer différentes méthodes de détection.
* Évaluer l’efficacité et le coût de calcul des algorithmes.

## Lien avec le thème

Ce projet est lié au thème **Sobriété – Efficacité – Optimisation** :

* **Sobriété** : concevoir un algorithme utilisant peu de ressources informatiques.
* **Efficacité** : détecter les anomalies rapidement et avec précision.
* **Optimisation** : trouver le meilleur compromis entre précision, temps de calcul et consommation de ressources.

## Méthodologie

Le projet reposera sur plusieurs étapes :

1. Étude du fonctionnement des réseaux informatiques.
2. Analyse statistique du trafic réseau.
3. Implémentation d’algorithmes de détection d’anomalies.
4. Simulation et comparaison des performances.

## Bibliographie

**NIST – Guide to Intrusion Detection and Prevention Systems (SP 800-94)**
https://nvlpubs.nist.gov/nistpubs/legacy/sp/nistspecialpublication800-94.pdf
Ce rapport présente les principes des systèmes de détection d’intrusion et les différentes méthodes de détection d’anomalies dans les réseaux.

**Andrew Tanenbaum – Computer Networks**
https://www.pearson.com/en-us/subject-catalog/p/computer-networks/P200000003521
Ouvrage de référence expliquant l’architecture et le fonctionnement des réseaux informatiques.

**Chandola, Banerjee, Kumar – Anomaly Detection: A Survey**
https://doi.org/10.1145/1541880.1541882
Article scientifique qui présente différentes méthodes de détection d’anomalies dans les données.

**Techniques de l’ingénieur – Cybersécurité et détection d’intrusion**
https://www.techniques-ingenieur.fr
Base documentaire technique décrivant les méthodes industrielles de surveillance et de sécurité des réseaux.
