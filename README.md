# TIPE-2026
Optimisation de la circulation de l'information dans un réseau


## Problématique
Comment déterminer le chemin optimal pour transmettre une information dans un réseau tout en minimisant le temps de transmission et l'utilisation des ressources ?

## Lien avec le thème
Ce projet s'inscrit dans le thème Sobriété – Efficacité – Optimisation.

- Sobriété : réduire l'utilisation des ressources du réseau.
- Efficacité : transmettre l'information rapidement.
- Optimisation : trouver le meilleur chemin dans un réseau.

## Méthode
Le réseau sera modélisé par un graphe.
Nous étudierons différents algorithmes comme :

- Dijkstra
- A*
- Flot maximal

Une simulation sera réalisée en Python pour comparer leurs performances.
